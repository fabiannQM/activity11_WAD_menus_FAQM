# Menus Project - Urban Travel & Exploration

## Description
This project demonstrates a multi-view web application built with Laravel, featuring a persistent navigation system with active-state highlighting. The theme is based on high-end, comfortable travel and vacation exploration.

## Features
- **Primary View**: Navigation menu with "Destinations", "Hidden Gems", "Travel Log", and "Bookings".
- **Active Selection**: Dynamic styling applied to the current menu option.
- **Universal Footer**: Visible across all views with activity/creator information.
- **Premium Design**: Modern CSS with glassmorphism, Google Fonts, and responsive layouts.
